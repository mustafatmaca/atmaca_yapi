# Atmaca Yapı Mobile App
Atmaca Yapı Mobile App is inventory tracking system. It is actively using by Atmaca Yapı Malzemeleri. It have add, delete, update and search functionality.

## Technologies
-Flutter
-GetX
-Firebase
